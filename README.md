# Bigkas Mobile

Bigkas Mobile is an Android-focused React Native (Expo) application that helps users practice public speaking and receive automated feedback on their vocal delivery. It captures high-quality audio on mobile, sends it to a backend for analysis, and shows practice results in a simple interface.

## Features

- Record speech practice sessions using the device microphone.
- Upload audio securely to the backend for processing.
- View basic practice information and navigation between core screens (Dashboard, Practice, History, Profile).
- Built with JavaScript (JSX) using Expo and React Navigation.

## Tech Stack

- **Framework:** React Native (Expo)
- **Language:** JavaScript (JSX)
- **Navigation:** React Navigation
- **Backend (external):** Python FastAPI
- **Package manager:** npm or yarn
